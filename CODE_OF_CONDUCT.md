# Code of Conduct

**Scope.** Applies to all project spaces (issues, PRs, discussions) and when representing the project.

**Standards.** Be professional. No harassment, hate or discrimination, sexualized content, doxxing, impersonation, threats, or sustained off-topic spam.

**Enforcement.** Maintainers may remove content and restrict participation. Sanctions: correction → warning → temporary restriction → ban.

**Reporting.** Report incidents privately via a maintainer’s GitHub profile (Daniel Thermons, Alex Girin, Vincent Brook) with links/timestamps. We acknowledge within 7 days. Security issues: see `SECURITY.md`.

**Attribution.** Based on the Contributor Covenant v2.1.
