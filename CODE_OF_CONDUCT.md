# Code of Conduct

This project follows the Contributor Covenant.

## Our Expectations
- Be respectful and inclusive. No harassment, hate speech, or personal attacks.
- Assume good intent, give constructive feedback, and keep discussions project-focused.
- Accept responsibility for mistakes and work to resolve conflicts in good faith.

## Scope
This Code applies across all project spaces (issues, PRs, discussions, chat) and when representing the project in public.

## Enforcement Responsibilities
Project maintainers clarify and enforce this Code, may remove or edit non-compliant content, and may warn or restrict contributors when needed.

## Enforcement Guidelines
- **Correction.** A private or public note requesting changes to problematic content.
- **Warning.** An official warning for a first or repeated violation.
- **Temporary Restriction.** Time-limited ban from participation for repeated violations.
- **Permanent Ban.** Removal from the community for severe or sustained misconduct.
